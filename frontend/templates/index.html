About FoodSight AI

🎯 Project Overview

FoodSight AI is an advanced deep learning application for Indian food classification and nutrition estimation. Using state-of-the-art computer vision, we help users make informed dietary choices by identifying dishes and providing comprehensive nutritional information.

🤖 AI Classification

100 Indian dishes with 91% accuracy

Complete macro & micronutrient data

🎨 Premium UI
